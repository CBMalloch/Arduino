\subsection{Detailed Description}
Wrapper for Audio Design Tool -\/\+Reverb-\/. 

See \href{https://www.pjrc.com/teensy/gui/}{\tt Paul Stoffregen\textquotesingle{}s Audio Design Tool}

I have added a 23\+L\+C1024 memory chip to the Open\+Effects Project hardware board where provision was made for it by Ã˜yvind Mjanger, the developer of the project. This chip allows for up to 1 1/2 seconds of delay to be added to the input signal, in each of 8 parallel channels. I\textquotesingle{}ve programmed only four of these.
