Detailed Description

Wrapper for Audio Design Tool -Reverb-.

See Paul Stoffregen's Audio Design Tool

I have added a 23LC1024 memory chip to the OpenEffects Project hardware board where provision was made for it by Øyvind Mjanger, the developer of the project. This chip allows for up to 1 1/2 seconds of delay to be added to the input signal, in each of 8 parallel channels. I've programmed only four of these.
