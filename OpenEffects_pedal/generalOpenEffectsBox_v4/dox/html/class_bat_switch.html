Detailed Description

Wrapper for hardware interface -BatSwitch-.

The bat switches are 3-position toggles, designated (ON)-OFF-(ON), meaning that the switch stays in the center position when it is put there, but it springs back when moved to either side position ( that's what the parentheses indicate ). The two side positions are connected to ground through two different valued resistors, so that the voltage is pulled down to a different value with each position of the switch.
