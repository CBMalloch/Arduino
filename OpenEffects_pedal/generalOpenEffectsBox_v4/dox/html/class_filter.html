Detailed Description

Wrapper for Audio Design Tool -Filter-.

See Paul Stoffregen's Audio Design Tool

Also see the Wikipedia article on Wah-wah pedals I am trying to create a wah-wah pedal using an expression pedal routed to control the base frequency of this filter.
